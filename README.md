mobx-react-demo
=====================
基于[MobX](https://mobxjs.github.io/mobx)，[React](https://facebook.github.io/react)，[react-router](https://reacttraining.com/react-router/web/api/HashRouter)，的demo。
打包工具：[webpack3](https://doc.webpack-china.org/)


### 目录结构

```
src             //项目目录
├── components  //组件
├── index.js    //入口文件
├── models      //状态
├── routes      //页面
├── services    //api
└── utils       //工具文件
```

### 命令

```
npm install
npm run start
```
